# Web-scraping

Scrape Viadeo with Puppeteer under Node and React.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine.<br/><br/>
git clone https://github.com/gsuiffet/webscraping.git

### Installing
```
npm install
```
```
npm start
```
